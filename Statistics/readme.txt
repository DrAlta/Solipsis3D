Example of statistics computation
By default, stats files are stored into "C:\Program Files\Solipsis"

Open 1 command window

Go into Statistics sub-directory
cd "Program Files\Solipsis\Statistics"

Generate CSV file from produced statistics
StatsGenerateCSV.py -p "C:\Program Files\Solipsis" -s 20090201000000 -e 20090228000000

Open resulting CSV file with Open Office Calc, Microsoft Excel, or any CSV file reader
