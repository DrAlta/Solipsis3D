HELP
--------------------------------------------------------------------------------

WorldsServer manages worlds hosted on RakNetServer running instances.
It also provide 1 simple user management by storing/giving nodeId per login.

WorldsServer.py is taking 1 optional arg:
-p <port:int>                   : port (default: 8550)


--------------------------------------------------------------------------------
HOW TO BUILD
--------------------------------------------------------------------------------

Refer to the Solipsis readme.txt

Navi.js was compressed with YUI compressor,
download http://www.julienlecomte.net/yuicompressor/yuicompressor-2.3.6.zip, unzip it
and run command java -jar yuicompressor-2.3.6.jar --type js Navi.js -o Navi-yui.js
